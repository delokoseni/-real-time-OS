\begin{frame}{OTP (Open Telecom Platform)}
	OTP предоставляет готовые механизмы устойчивости:
	
	\begin{itemize}
		\item Стандартизированные процессы;
		\item Стратегии восстановления:
		\begin{itemize}
			\item one\_for\_one
			\item one\_for\_all
			\item rest\_for\_one
		\end{itemize}
		\item Шаблоны распределённых систем.
	\end{itemize}
\end{frame}
